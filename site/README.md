# 个人空间前后台

## site-admin 后台管理系统

默认配置在src/config目录

安装依赖：
```shell
npm i
```

启动项目：
```shell
npm run dev
```

打包项目

```shell
npm run build:prod
```


## site-client 前台

安装依赖：
```shell
npm i
```

启动项目：
```shell
npm run serve
```

打包项目

```shell
npm run build
```


## site-serer 服务器
默认监听端口为 9527
默认配置在config文件夹

安装依赖：
```shell
npm i
```

启动项目：
```shell
npm start
```

API.md介绍了数据库的表和部分API响应
