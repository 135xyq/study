# ==和===操作符的区别

## 等于操作符

==(等于符)在进行比较时, 如果两边操作数的数据类型不同时会先进行类型转化(自动完成)

- 当一方操作数为布尔值时:

    js会将两操作数都转化为数字后再进行比较

- 当一方为对象,另一方为数字时:

    js将对象转化为数字(参考Number(object)的转化过程), 然后再进行比较

## 全等于操作符

===(全等符)在进行比较时, 不会进行类型转化

- 当两操作数类型不一致时, 直接返回false
- 一致时:
  - 操作数为原始值时: 要求两操作数必须相等时才返回true
  - 操作符为引用值时: 比较地址
